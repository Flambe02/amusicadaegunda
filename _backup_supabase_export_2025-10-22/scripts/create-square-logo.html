Générateur d'icônes carrées iOS - Música da Segunda

Instructions :

1. Cliquez sur "Générer les icônes" pour créer les icônes carrées basées sur l'image originale

2. Cliquez sur "Télécharger" pour chaque taille d'icône

3. Remplacez les icônes existantes dans public/images/

4. Les icônes apple-touch-icon-precomposed forcent l'affichage carré sur iOS
